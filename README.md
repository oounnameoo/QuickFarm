# QuickFarm

[![Build](https://github.com/YOUR_USERNAME/quickfarm/actions/workflows/build.yml/badge.svg)](https://github.com/YOUR_USERNAME/quickfarm/actions/workflows/build.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A lightweight [Paper](https://papermc.io/) plugin that lets you **harvest and instantly replant** crops with a single right-click — no commands, no config, no fuss.

---

## Features

- **One-click harvest & replant** — right-click a fully grown crop while holding the matching seed to harvest it and immediately reset the age back to 0.
- **Natural loot tables** — drops are generated by Bukkit's `getDrops()` API, so Fortune enchantments and vanilla loot tables apply correctly.
- **Seed-gated** — you must be holding the correct seed in your main hand; nothing happens if you're holding the wrong item or empty hand.
- **Farmland safe** — the soil block is never broken or disturbed; only the crop is reset.
- **Satisfying feedback** — plays the vanilla `BLOCK_CROP_BREAK` sound on every harvest.
- **Zero configuration** — drop it in and it works.

---

## Supported Crops

| Crop block | Required seed in hand |
|---|---|
| Wheat | Wheat Seeds |
| Carrots | Carrot |
| Potatoes | Potato |
| Beetroots | Beetroot Seeds |
| Nether Wart | Nether Wart |

---

## How to Use

1. Hold the correct seed for the crop in your **main hand**.
2. Find a **fully grown** crop.
3. **Right-click** the crop block.
4. The crop drops its harvest on the ground and is instantly replanted using 1 seed from your hand.

> **Tip:** If the crop isn't fully grown, nothing happens and the interaction is ignored.

## Installation

1. Download the latest `quickfarm-*.jar` from the [Releases](https://github.com/YOUR_USERNAME/quickfarm/releases) page.
2. Place the JAR in your server's `plugins/` directory.
3. Restart (or `/reload confirm`) your server.
4. No configuration needed — the plugin is ready to use.

---

## Building from Source

```bash
git clone https://github.com/YOUR_USERNAME/quickfarm.git
cd quickfarm
mvn package
```

The built JAR will be at `target/quickfarm-1.0.0.jar`.
